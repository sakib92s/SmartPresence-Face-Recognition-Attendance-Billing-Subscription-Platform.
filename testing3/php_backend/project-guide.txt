After attendance is recorded, the system provides a Wage Muster Generation feature to calculate employee wages based on attendance data.

From the dashboard, HR/Admin can open the Wage Muster section.

The HR can:

Select or upload the attendance Excel file generated by the Python system

Click on Generate Wage Muster

The system processes the file using the wage muster module and automatically:

Reads attendance data

Calculates total working days and present days

Calculates basic wages and salary components

Generates a formatted Wage Muster Excel file

The generated Wage Muster file is saved in:

php_backend/wage_muster_generator/generated/


The HR can then download the Wage Muster report using the web portal.

This helps HR and payroll teams use the report directly for salary processing.
